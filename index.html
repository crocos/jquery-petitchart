EXPERIMENTAL

Currently not yet released.

Example

» Bar Chart

<div data-petitchart="120,300,50,100,500,200,10,30,90"></div>
<div data-petitchart="120,300,50,100,500,200,10,30,90" data-petitchart-color="blue" data-petitchart-width="320" data-petitchart-height="120"></div>

» Line Chart

<div data-petitchart="120,300,50,100,500,200,10,30,90" data-petitchart-type="line"></div>
<div data-petitchart="120,300,50,100,500,200,10,30,90" data-petitchart-color="blue" data-petitchart-width="320" data-petitchart-height="120" data-petitchart-type="line"></div>

Installation

Include script after the jQuery library.

<script src="dist/jquery.petitchart.min.js"></script>

Do not include the script directly from GitHub (http://raw.github.com/...). The file is being served as text/plain and as such being blocked in Internet Explorer on Windows 7 for instance (because of the wrong MIME type). GitHub is not a CDN.
